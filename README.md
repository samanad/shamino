## üé® Customization

### Colors
The website uses CSS custom properties for easy color customization. Edit the `:root` section in `styles.css`:

```css
:root {
    --primary-red: #ff2d55;
    --primary-orange: #ff6b35;
    --primary-yellow: #ffd700;
    /* ... more colors */
}
```
